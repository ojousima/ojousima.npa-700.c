◆ npa_ret_t

typedef int8_t npa_ret_t

Error code from NPA 700 driver.

Negative values are fatal errors, positive values are warnings.

Value | Meaning

0 | Success. -1 | Error: I2C interface function returned NACK. -2 | Error: I2C interface function timed out. -4 | Error: NULL pointer supplied. -8 | Error: Function is not implemented. 1 | Warning: Value is saturated. 2 | Warning: Value is already read.
